Setup

Download the SnippetDrop extension and login with GitHub. A public/private key pair will be generated locally.

Send

Right-click any selected text or file and select Send via SnippetDrop. Enter the recipient's GitHub username.

Receive

View all received snippets in your SnippetDrop Inbox. Also view past sent snippets and block unwanted senders.

End-to-End Encryption

SnippetDrop uses 2048 bit asymmetric RSA encryption via node-rsa.

Key pairs are generated locally on your device.

Private keys stay on your device; never uploaded. Public keys are uploaded to SnippetDrop's server.

Sending a snippet: 1) extension downloads recipient's public keys, 2) encrypts snippet, 3) uploads encrypted snippet to SnippetDrop's servers.

Receiving snippets: 1) extension fetches encrypted snippets for your username + public key, 2) decrypts and stores them locally using private key.

Encrypted snippets are deleted from servers once the recipient has downloaded them or 7 days has passed.

Native support for multiple devices per user. Snippets are encrypted and sent per device public key.
